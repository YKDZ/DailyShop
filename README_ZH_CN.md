## 文档

- [ ] 基于市场需求的动态定价
- [ ] 基岩版菜单
- [ ] 商店商品总量
- [x] 交易日志
- [ ] SQL 支持
- [ ] 随机数量以及基于数量的定价

- [ ] 打折
